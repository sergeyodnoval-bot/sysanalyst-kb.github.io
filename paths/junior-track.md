# Путь Junior System Analyst

Этот трек проведёт вас от базовых понятий до понимания REST API и OpenAPI.
Рекомендованная последовательность:

1. **Клиент-серверная архитектура** — фундаментальное понятие
2. **HTTP** — протокол, на котором всё построено
3. **BPMN** — моделирование процессов
4. **REST API** — практическая интеграция
5. **OpenAPI** — документирование API
