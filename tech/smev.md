СМЭВ — государственная «шина» для обмена данными между ГИС.

**Версии:**
- **СМЭВ 2** — SOAP/XML, УКЭП, синхронный, ViPNet Coordinator
- **СМЭВ 3** — REST/JSON, OAuth 2.0 + УКЭП, асинхронный, любой шлюз

**Ключевые понятия:**
- **Электронный сервис** — API, опубликованный ГИС-поставщиком
- **Виды сведений (ВС)** — тип данных для обмена
- **Метаданные** — описание сервиса в реестре
- **Шлюз СМЭВ** — компонент для подключения к СМЭВ

**Для аналитика:** при проектировании ГИС нужно определить, какие виды сведений запрашивать через СМЭВ, какие — предоставлять. В спецификации указать: транспорт (REST), формат (JSON), SLA (5–30 сек), тип (sync/async).
