# Large Language Model (LLM)

**LLM** — нейросетевая модель, обученная предсказывать следующую единицу текста (токен) на огромных корпусах данных. Современные LLM (GPT-4, Claude, Llama, YandexGPT) способны генерировать связный текст, отвечать на вопросы, писать код и выполнять инструкции.

## Для чего используется

- **Чат-боты и ассистенты** — генерация ответов на естественном языке
- **RAG-системы** — генерация ответа с опорой на внешнюю базу знаний
- **Извлечение и суммаризация** — обработка больших объёмов текста
- **Генерация кода** — автодополнение, ревью, документация
- **Классификация текста** — тональность, тематика, интент

## Ключевые концепции

- **Transformer** — архитектура нейросети, лежащая в основе всех современных LLM (Google, 2017)
- **Tokenization** — разбиение текста на токены (не слова, а подслова)
- **Context window** — максимальный объём текста, который модель «видит» за один запрос
- **Attention** — механизм, позволяющий модели учитывать взаимосвязи между всеми токенами в контексте
- **Fine-tuning** — дообучение предобученной модели на специфическом датасете

## Когда использовать

- Задача требует понимания естественного языка
- Нужна генерация связного текста, а не просто классификация
- Возможность occasional errors acceptable (некритичные галлюцинации)
- Есть бюджет на инференс (особенно для больших моделей)

## Когда НЕ использовать

- Требуется 100% точность и детерминированность (бухгалтерия, расчёты)
- Критична скорость — LLM медленнее классических ML-моделей
- Нет бюджета на GPU / API — стоимость инференса LLM выше
- Нужна работа с числами — LLM плохо справляются с арифметикой

## Альтернативы

| Альтернатива | Когда выбрать |
|-------------|---------------|
| Классические ML-модели (Random Forest, Logistic Regression) | Классификация, регрессия — быстрее и дешевле |
| Rule-based системы | Если бизнес-правила известны и стабильны |
| Search + snippets | Если не нужна генерация, достаточно найти ответ в базе |
| Smaller specialized models (BERT, T5) | Если задача узкая, можно сделать быстрее и дешевле |

## Как начать

- [HuggingFace](https://huggingface.co/) — библиотека transformers, тысячи предобученных моделей
- [OpenAI API](https://platform.openai.com/) — SaaS для GPT-моделей
- [Ollama](https://ollama.ai/) — запуск open-source LLM локально
- [LangChain](https://www.langchain.com/) — фреймворк для LLM-приложений
