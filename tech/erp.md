ERP (Enterprise Resource Planning) — система управления ресурсами предприятия: финансы, закупки, производство, склад, персонал.

**Ключевые модули ERP для e-commerce:**
- **FI (Finance)** — бухгалтерский учёт, проводки
- **CO (Controlling)** — себестоимость, бюджет
- **MM (Materials Management)** — закупки, запасы
- **SD (Sales & Distribution)** — продажи, отгрузки
- **WM (Warehouse Management)** — складской учёт (часто отдельно, WMS)

**Интеграция OMS → ERP:**
- Заказ → отгрузка → проводка в FI
- Поступление товара → учёт в MM
- Возврат → сторно проводки

**Для аналитика:** ERP — система учёта (а не операционного контура). OMS создаёт заказ, ERP отражает его в бухгалтерии. Интеграция — через REST / EDI / 1С-разъёмы. Ключевое: синхронность данных между OMS и ERP (расхождения ведут к проблемам с отчётностью).
