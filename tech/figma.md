# Figma

Figma — это инструмент для дизайна интерфейсов и прототипирования. В нём работают дизайнеры, но и аналитику полезно знать основы — чтобы понимать, как устроены макеты, и уметь смотреть их.

## Что в нём делают

- **Рисуют макеты** — как будет выглядеть сайт или приложение
- **Создают прототипы** — интерактивные модели, по которым можно «походить»
- **Работают совместно** — несколько человек могут редактировать один файл одновременно
- **Передают в разработку** — программисты смотрят размеры, цвета, отступы прямо в макете

## Зачем это аналитику

Аналитик не рисует макеты (это делает дизайнер), но:

- Нужно понимать, как читать макеты и давать обратную связь
- Иногда аналитик делает простые wireframes (черновые наброски) для обсуждения идей
- При проектировании API или логики полезно видеть, как это будет выглядеть для пользователя

Figma бесплатна для личного использования — зарегистрируйтесь и попробуйте.
