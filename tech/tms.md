:::info[TL;DR]
TMS (Transportation Management System) — класс систем для управления транспортом и перевозками. Ключевые вендоры: 1С:TMS, SAP TM, Oracle OTM, Solomon TMS, RetaiLite. В РФ распространены 1С:TMS, Solomon и собственные разработки крупных ритейлеров.
:::

## Функции TMS

| Функция | Описание |
|---------|----------|
| **Планирование** | Маршруты, расписание, консолидация грузов |
| **Управление перевозчиками** | Тендеры, договоры, рейтинг |
| **Тарификация** | Base rate, надбавки, топливный сбор |
| **Трекинг** | GPS, статусы, ETA |
| **ЭДО** | УПД, ЭТРН, счета |
| **Взаиморасчёты** | Акты, сверка, оплата |

## Популярные TMS в РФ

| Система | Платформа | Особенность |
|---------|-----------|-------------|
| **1С:TMS** | 1С | Интеграция с 1С:ERP, 1С:Бухгалтерия |
| **Solomon** | SaaS | Ритейл, FMCG |
| **RetaiLite** | SaaS | Оптимизация маршрутов |
| **SAP TM** | SAP | Enterprise |
| **Oracle OTM** | Oracle | Enterprise |
