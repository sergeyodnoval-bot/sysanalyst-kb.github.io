Astra Linux — отечественная ОС на базе Debian. Основная ОС для ГИС в рамках импортозамещения.

**Редакции:**
- **Common Edition** — общего назначения (для серверов и рабочих станций)
- **Special Edition** — для ИС с грифом «секретно» и «особой важности»

**Ключевые возможности:**
- Сертификация ФСТЭК (УЗ-1, УЗ-2, УЗ-3)
- Защищённая среда (пакеты защищённого доступа)
- Parsec — система мандатного доступа
- Поддержка ГОСТ-криптографии (через КриптоПро)

**Для аналитика:** при выборе ОС для ГИС указать в спецификации: Astra Linux Special Edition (версия 1.7+), сертифицированная ФСТЭК. Для обычных госучреждений — Common Edition. Совместимость с Postgres Pro и «1С» — подтверждена.
