# Miro

Miro (ранее RealtimeBoard) — это онлайн-доска для совместной работы. Аналитики используют её для воркшопов, мозгоштурмов, картографирования процессов и визуализации идей с заказчиком.

## Для чего используется

- **Event Storming** — моделирование доменных событий
- **User Story Mapping** — дорожная карта пользовательских историй
- **Impact Mapping** — связь бизнес-целей с функциональностью
- **Retrospectives** — ретроспективы (в том числе распределённые команды)
- **Customer Journey Map** — карта пути пользователя
- **Сбор идей на воркшопах** — стикеры, голосование, кластеризация

## Ключевые концепции

### Основные элементы

| Элемент | Назначение |
|---------|-----------|
| Стикеры | Идеи, задачи, вопросы (цвет = категория) |
| Фреймы | Границы разделов/экранов |
| Шаблоны | Готовые доски: CJM, ES, USM |
| Таймер | Контроль времени на воркшопе |
| Voting | Голосование участников |

### Форматы воркшопов

| Тип | Участники | Длительность | Результат |
|-----|-----------|-------------|-----------|
| Event Storming | Команда + эксперты | 2-4 часа | Карта доменных событий |
| User Story Mapping | PO + аналитик + dev | 1-2 часа | Дорожная карта US |
| Retro | Вся команда | 30-60 мин | План улучшений |
| CJM Workshop | Заказчик + аналитик | 2-3 часа | CJM с болями и идеями |

### Miro в России

В 2022 году Miro объявил об уходе из РФ. Российские компании переходят на:
- **Sboard** — российский аналог
- **Weje** — альтернатива
- **Dictel** — экосистема Ростелекома

## Когда использовать

- Нужен воркшоп с заказчиком (очный или удалённый)
- Команда работает распределённо
- Нужна визуализация процессов для совместного обсуждения
- Сбор и кластеризация идей

## Как начать

1. Зарегистрируйтесь на [miro.com](https://miro.com/) (бесплатно до 3 досок)
2. Откройте шаблон «Event Storming» или «User Story Mapping»
3. Пригласите коллег по ссылке
4. Начните с простого: расставьте стикеры, сгруппируйте, обсудите
