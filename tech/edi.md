:::info[TL;DR]
EDI (Electronic Data Interchange) — стандарт обмена структурированными документами между участниками логистической цепочки. В РФ используется EDI-провайдеры (Диадок, СБИС, 1С-ЭДО, Корус) и форматы: EDIFACT, XML, JSON. Типы сообщений: ORDERS (заказ), DESADV (уведомление об отгрузке), INVOIC (счёт).
:::

## Типы EDI-сообщений в логистике

| EDI-сообщение | Описание |
|---------------|----------|
| **ORDERS** | Заказ на доставку |
| **DESADV** | Уведомление об отгрузке |
| **RECADV** | Подтверждение получения |
| **INVOIC** | Счёт-фактура |
| **REMADV** | Уведомление об оплате |
| **IFTMIN** | Инструкция по перевозке |
| **IFTSTA** | Статус перевозки |
