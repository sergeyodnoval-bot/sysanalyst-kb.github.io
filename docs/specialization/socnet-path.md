:::info[TL;DR]
Аналитик соцсетей и контентных платформ работает с лентой (feed), мессенджерами, модерацией, социальным графом, монетизацией (реклама/подписки), контентной платформой (видео, музыка, stories) и API/SDK для внешних разработчиков. Ключевые вызовы: масштаб (миллиарды пользователей), real-time, ASR/NSFW-модерация, рекомендации.
:::

## Основные домены

```mermaid
flowchart TD
    PLATFORM["Соцсеть / Платформа"]
    PLATFORM --> FEED["Лента контента<br/>— ranking, рекомендации"]

    PLATFORM --> MOD["Модерация<br/>— ASR, NSFW, фрод"]
    PLATFORM --> GRAPH["Социальный граф<br/>— подписки, друзья"]
    PLATFORM --> MON["Монетизация<br/>— реклама, подписки"]
    PLATFORM --> MEDIA["Контент-платформа<br/>— видео, музыка, stories"]
    PLATFORM --> API["API / SDK / Маркетплейс"]
```
