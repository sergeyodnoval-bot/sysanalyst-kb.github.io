:::info[TL;DR]
Медицина — одна из самых регулируемых отраслей. 323-ФЗ (об основах охраны здоровья), 152-ФЗ (ПД особой категории), приказы Минздрава, регистрация медизделий. Все данные о здоровье — ПД особой категории (усиленная защита). Любая МИС должна соответствовать требованиям к обработке ПД и, в ряде случаев, регистрироваться как медизделие.
:::

## Основные НПА

| НПА | Описание |
|-----|----------|
| **323-ФЗ** | Об основах охраны здоровья: права пациента, ЭМК, телемедицина |
| **152-ФЗ** | ПД особой категории (здоровье) — УЗ-1, согласие, локализация |
| **Приказ Минздрава № 947н** | Правила ведения ЭМК |
| **Приказ № 797н** | Телемедицина — порядок оказания |
| **Постановление № 1275** | ЕГИСЗ — передача данных |

## ПД особой категории

| Требование | Описание |
|------------|----------|
| **Согласие** | Отдельное согласие на обработку ПД о здоровье |
| **УЗ-1** | Максимальный уровень защищённости |
| **Локализация** | Серверы в РФ |
| **Доступ** | RBAC + аудит каждого доступа |
| **Шифрование** | Каналы + диски (ГОСТ) |
| **Уничтожение** | При отзыве согласия |

## Регистрация медизделия

Если МИС или модуль влияет на принятие врачебных решений — он считается медизделием:

| Класс | Риск | Пример | Регистрация |
|-------|------|--------|-------------|
| **1** | Низкий | Архив ЭМК (только просмотр) | Уведомление |
| **2a** | Средний | МИС с диагностикой | Регистрация Росздравнадзор |
| **2b** | Повышенный | Системы поддержки принятия решений | Регистрация + КИ |
| **3** | Высокий | ИИ-алгоритмы диагностики | Регистрация + КИ |

## Требования к системе (спецификация)

| Параметр | Пример |
|----------|--------|
| Класс ПД | Особой категории (здоровье) |
| Согласие | Электронное, через Госуслуги или ЕСИА |
| Аудит | Все действия с ЭМК, хранение 10 лет |
| УКЭП | Врач подписывает записи |
| Медизделие | Регистрация (2a/2b) если влияет на диагноз |
| ЕГИСЗ | Передача по HL7 FHIR |

## Проверь себя

1. **Какие законы регулируют медицину в IT?**
   *Ответ:* 323-ФЗ (медицина), 152-ФЗ (ПД), приказы Минздрава, постановления о ЕГИСЗ.

2. **Когда МИС считается медизделием?**
   *Ответ:* Если она влияет на принятие врачебных решений (диагностика, лечение). Класс риска — от 2a до 3.
