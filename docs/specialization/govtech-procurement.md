:::info[TL;DR]
44-ФЗ — госзакупки для бюджетных учреждений. 223-ФЗ — для госкомпаний и естественных монополий. Аналитик на стороне заказчика пишет ТЗ (техническое задание) для тендера. На стороне исполнителя — готовит техпредложение и оценивает объём работ.
:::

## 44-ФЗ vs 223-ФЗ

| Параметр | 44-ФЗ | 223-ФЗ |
|----------|-------|--------|
| **Заказчики** | Бюджетники, казённые учреждения | Госкомпании, монополии |
| **Процедура** | Строгая (аукцион, конкурс) | Гибче |
| **Сроки** | 30–90 дней | 15–45 дней |
| **ТЗ** | Обязательно, жёсткие требования | Есть, но гибче |
| **Участники** | Любые (СМП — квота 15%) | Определённые заказчиком |

## Процесс закупки (44-ФЗ)

```mermaid
flowchart LR
    P["Планирование"] --> RS["Разработка ТЗ"]
    RS --> PUB["Публикация в ЕИС"]
    PUB --> BID["Приём заявок"]
    BID --> WIN["Определение победителя"]
    WIN --> CONT["Заключение контракта"]
    CONT --> EXEC["Исполнение"]
    EXEC --> ACC["Приёмка и оплата"]
```

## Задачи аналитика на закупке

### На стороне заказчика:
- Сбор и формализация требований
- Написание ТЗ для тендера (техническая часть)
- Проверка заявок участников на соответствие ТЗ
- Участие в приёмке

### На стороне исполнителя:
- Анализ ТЗ, оценка объёмов
- Подготовка техпредложения
- Проектирование решения
- Обоснование НМЦК (начальной цены)

## Структура ТЗ на IT-систему

| Раздел | Содержание |
|--------|-----------|
| **Общие сведения** | Наименование, цели, сроки |
| **Назначение и цели** | Для чего система |
| **Функциональные требования** | Что должна делать |
| **Требования к архитектуре** | Состав подсистем, интеграции |
| **Требования к безопасности** | ФСТЭК, СЗИ, аттестация |
| **Требования к импортозамещению** | Реестр ПО |
| **Состав и содержание работ** | Этапы |
| **Порядок сдачи и приёмки** | Критерии |

## Проверь себя

1. **Чем 44-ФЗ отличается от 223-ФЗ?**
   *Ответ:* 44-ФЗ — строгий, для бюджетников. 223-ФЗ — гибче, для госкомпаний.

2. **Какие разделы обязательно включить в ТЗ?**
   *Ответ:* Функциональные требования, архитектура, безопасность, импортозамещение, состав работ, приёмка.
