# Что такое soft skills и зачем они аналитику

Soft skills — это навыки, не связанные с конкретными технологиями или инструментами. Они определяют, как вы работаете с людьми, информацией и собой.

## Почему для аналитика это критично

Системный аналитик — мост между бизнесом и разработкой. Без soft skills этот мост рушится:

- **Стейкхолдер** говорит на языке бизнеса — аналитик должен перевести это на язык требований
- **Разработчик** задаёт уточняющие вопросы — аналитик должен терпеливо объяснить
- **Команда** не понимает, зачем нужна фича — аналитик должен донести ценность

Технические навыки можно наработать за полгода. Soft skills — за годы.

## Ключевые soft skills для SA

**Коммуникация.** Умение задавать вопросы, слушать, переформулировать, доносить сложное простыми словами.

**Критическое мышление.** Умение сомневаться, проверять факты, видеть причинно-следственные связи.

**Эмпатия.** Понимание мотивов и болей собеседника — стейкхолдера, разработчика, пользователя.

**Фасилитация.** Умение управлять групповым обсуждением: держать фокус, укладываться в тайминг, собирать решения.

**Письменная коммуникация.** Чёткие и понятные документы, которые не нужно перечитывать дважды.

**Работа с конфликтами.** Умение не принимать на личный счёт, искать компромисс, переводить спор в конструктивное русло.

## Hard skills vs Soft skills

| Hard skills | Soft skills |
|------------|-------------|
| Знание SQL | Умение понять, что на самом деле нужно бизнесу |
| Умение рисовать BPMN | Умение провести workshop с 10 стейкхолдерами |
| Знание REST API | Умение объяснить разработчику, зачем нужен этот endpoint |
| OpenAPI | Умение настоять на своём решении, не испортив отношения |

## Как развивать

- **Практика.** Каждая встреча, каждое письмо — возможность улучшить навык.
- **Feedback.** Просите коллег оценить вашу коммуникацию.
- **Рефлексия.** После встречи спрашивайте себя: что прошло хорошо, что можно улучшить?

## Что дальше

- **Критическое мышление** — как проверять факты и избегать ошибок
- **Деловая переписка** — как писать понятные письма и документы
