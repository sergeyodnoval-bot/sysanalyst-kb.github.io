# Что такое интернет и компьютерная сеть

:::note
Компьютерная сеть — это несколько компьютеров, соединённых друг с другом, чтобы обмениваться данными. Интернет — это «сеть сетей», самая большая компьютерная сеть на планете.
:::

Представьте, что вам нужно отправить письмо из Москвы во Владивосток. Вы кладёте лист в конверт, пишете адрес, несёте на почту. Письмо едет на поезде, потом на машине, проходит через несколько сортировочных центров и в итоге попадает в почтовый ящик адресата.

Интернет работает так же. Только вместо бумажных писем — **пакеты данных**, вместо адреса — **IP-адрес**, вместо почтовых отделений — **маршрутизаторы (роутеры)**, а вместо почтальона — **протоколы**.

## Что такое компьютерная сеть

Компьютерная сеть — это два или более компьютеров, соединённых кабелем или по Wi-Fi, чтобы обмениваться данными.

Самые простые сети:

- **Домашняя сеть** — ваш компьютер, телефон, телевизор подключены к одному роутеру
- **Офисная сеть (LAN)** — компьютеры сотрудников соединены в локальную сеть для общего доступа к принтеру и файлам
- **Глобальная сеть (WAN)** — сеть, соединяющая города и страны. Интернет — это огромная WAN

## Как работает интернет

Интернет — это не один кабель, а миллионы сетей, соединённых друг с другом. Когда вы открываете сайт, происходит примерно следующее:

1. **Ваш компьютер** отправляет запрос: «покажи мне страницу example.com»
2. **Роутер** в вашей квартире передаёт запрос провайдеру (вашему «почтовому отделению»)
3. **Провайдер** направляет запрос через несколько промежуточных узлов — как письмо через сортировочные центры
4. **Сервер** example.com получает запрос и отправляет обратно страницу сайта
5. Страница возвращается к вам тем же путём

Весь процесс занимает доли секунды.

## IP-адреса — адреса в сети

У каждого устройства в сети есть уникальный **IP-адрес** — как почтовый адрес дома.

**IPv4** — старый формат, выглядит как четыре числа через точку:
`192.168.1.1`

**IPv6** — новый формат, выглядит длиннее:
`2001:0db8:85a3:0000:0000:8a2e:0370:7334`

Пример из жизни: когда вы заказываете пиццу, вы называете адрес доставки. Когда компьютер отправляет запрос, он указывает IP-адрес получателя.

**Специальные адреса:**
- `127.0.0.1` (localhost) — «мой собственный компьютер»
- `192.168.x.x` — адреса для домашних сетей (не выходят в интернет)

## DNS — телефонная книга интернета

Запоминать IP-адреса неудобно. Вы же не запоминаете номер телефона каждого сайта (хотя у них тоже есть IP).

**DNS (Domain Name System)** — это телефонная книга интернета. Когда вы вводите `google.com`, DNS-сервер говорит: «Google живёт по адресу 142.250.185.78».

Без DNS интернет был бы списком чисел — как если бы все сайты открывались только по IP-адресам.

## Порты — квартиры в доме

Если IP-адрес — это дом, то **порт** — это номер квартиры. Компьютер может одновременно делать много дел: показывать сайт (порт 80), отправлять почту (порт 25), принимать файлы (порт 21).

Стандартные порты:
- **80** — HTTP (веб-сайты)
- **443** — HTTPS (защищённые сайты)
- **22** — SSH (удалённый доступ)
- **5432** — PostgreSQL (база данных)

Когда вы открываете сайт, браузер сам знает, что нужно обращаться на порт 80 или 443.

## Почему это важно для аналитика

Понимание сети — база для работы с любыми интеграциями:

- **API** — запросы и ответы ходят по сети. Если API не работает — проблема может быть в сети, а не в коде
- **Клиент-сервер** — это сетевая архитектура. Нет сети — нет клиент-сервера
- **Интеграции** — любая интеграция двух систем включает сетевое взаимодействие
- **Протоколы** — HTTP, TCP/IP, WebSocket — все это сетевые протоколы

## Ключевые термины

| Термин | Значение |
|--------|---------|
| **Сеть** | Группа соединённых устройств для обмена данными |
| **Интернет** | Глобальная сеть, соединяющая миллионы частных сетей |
| **IP-адрес** | Уникальный адрес устройства в сети |
| **DNS** | Система, превращающая доменные имена в IP-адреса |
| **Порт** | Номер «квартиры» на устройстве для разных сервисов |
| **Роутер** | Устройство, направляющее трафик между сетями |
| **Пакет** | Единица данных, передаваемая по сети |

## Проверь себя

**1. Что произойдёт, если DNS-сервер перестанет работать?**

Вы не сможете открыть сайт по имени (например, google.com), но сможете — по IP-адресу. Если вы его знаете, конечно.

**2. Зачем нужны порты, если есть IP-адрес?**

IP-адрес находит устройство (дом). Порт указывает, какое приложение на этом устройстве должно получить данные (квартиру). Без портов компьютер не знал бы, какому приложению передать входящий запрос.

**3. Чем отличается локальная сеть от интернета?**

Локальная сеть (LAN) соединяет устройства в одном здании или квартире. Интернет — это множество локальных сетей, соединённых через провайдеров и магистральные каналы. Устройства в локальной сети видят друг друга напрямую; через интернет — только через маршрутизаторы.
