# Что такое программное обеспечение

*TL;DR: Программное обеспечение — это инструкции для компьютера, написанные на понятном ему языке. Без софта компьютер — просто кусок металла и пластика.*

Представьте, что вы купили самую мощную кухонную плиту в мире. Она может нагреваться до любых температур, имеет десятки режимов, таймеры, датчики. Но без рецепта эта плита бесполезна — она не знает, что именно готовить.

Компьютер — это такая же "плита". Сам по себе он может выполнять только простейшие операции: складывать числа, перемещать данные, включать/выключать сигналы. Чтобы компьютер сделал что-то полезное — показал видео, отправил письмо, рассчитал зарплату — ему нужны **инструкции**. Эти инструкции и есть программное обеспечение.

## Что входит в программное обеспечение

ПО состоит из двух частей:

**Код** — это текст инструкций, написанный на специальном языке программирования. Например: "если пользователь нажал кнопку, покажи окно".

**Данные** — это информация, с которой работает программа. Список товаров в интернет-магазине, текст вашего сообщения в мессенджере, фото в галерее.

Без кода компьютер не знает, что делать. Без данных ему нечего обрабатывать. Вместе код и данные создают программу, которая решает конкретную задачу.

## Типы программного обеспечения

**Системное ПО** — это фундамент. Операционная система (Windows, macOS, Linux) управляет железом компьютера и предоставляет базовые сервисы для других программ. Без неё компьютер не включится.

**Прикладное ПО** — это программы для конкретных задач. Браузер для интернета, Word для документов, Photoshop для картинок. Когда вы "устанавливаете программу" — вы ставите именно прикладное ПО.

**Инструменты разработки** — это программы для создания других программ. Редакторы кода, компиляторы, системы контроля версий. Это ПО для программистов.

## Как ПО связано с железом

Компьютер состоит из процессора, памяти, диска и периферии (экран, клавиатура, мышь). ПО — это посредник между вами и этим железом.

Когда вы кликаете мышкой, операционная система получает сигнал от мыши, понимает, что вы кликнули на иконку браузера, загружает браузер с диска в память, и браузер начинает выполнять свои инструкции. Вы видите на экране значок, но под капотом происходит десятки тысяч операций.

## Почему это важно знать

Системному аналитику не нужно уметь писать код (хотя это полезно). Но нужно понимать, что:

- Любая программа — это набор инструкций и данных
- Программы работают на конкретном железе с конкретными ограничениями
- Разные типы ПО решают разные задачи
- Без ПО компьютер бесполезен, а без железа ПО негде выполнять

Это понимание поможет вам говорить с программистами на одном языке и понимать, почему некоторые требования реализовать сложно или невозможно.

## Ключевые термины

- **Программное обеспечение (ПО)** — программы и данные для компьютера
- **Код** — инструкции, написанные на языке программирования
- **Операционная система** — системное ПО, управляющее железом
- **Прикладное ПО** — программы для конкретных задач пользователей

## Что дальше

Теперь, когда вы понимаете, что такое ПО, давайте разберёмся, как компьютер его выполняет. Следующая статья — [Как работает компьютер](/docs/basics/how-computer-works).

## Проверь себя

1. Что произойдёт, если включить компьютер без операционной системы?
2. В чём разница между системным и прикладным ПО?
3. Почему код без данных бесполезен?

Ответы: 1) Компьютер не загрузится — ему нужны инструкции. 2) Системное ПО управляет железом, прикладное решает задачи пользователя. 3) Код — это инструкции, а данные — то, что обрабатывается. Без данных нечего обрабатывать.
