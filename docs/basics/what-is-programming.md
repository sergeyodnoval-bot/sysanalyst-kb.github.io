# Что такое программирование

*TL;DR: Программирование — это перевод человеческих задач на язык, который понимает компьютер. Программист пишет инструкции, а компьютер их выполняет.*

Представьте, что вы приехали в страну, где говорят на незнакомом языке. Вы хотите купить хлеб, но не знаете слов. Вам нужен переводчик, который переведёт ваши слова на местный язык.

Программирование — это такой переводчик. Вы (человек) хотите, чтобы компьютер сделал что-то полезное: показал карту, отправил письмо, рассчитал зарплату. Но компьютер не понимает по-русски или по-английски. Ему нужен специальный язык — **язык программирования**.

Программист — это переводчик. Он берёт человеческую задачу и переводит её на язык, который понимает компьютер.

## Языки программирования

Существуют сотни языков программирования: Python, JavaScript, Java, C++, Go и многие другие. Каждый имеет свои особенности и применяется для разных задач.

Но все они решают одну задачу — дать компьютеру инструкции. Разница в том, как эти инструкции записываются.

Например, чтобы вывести на экран «Привет, мир!», на Python это одна строка:

```python
print("Привет, мир!")
```

А на Java — целая программа:

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Привет, мир!");
    }
}
```

Результат одинаковый, но синтаксис разный. Это как сказать «привет» по-русски или «hello» по-английски — смысл тот же, слова разные.

## Как программа превращается в машинный код

Компьютер не понимает Python или Java. Он понимает только **машинный код** — последовательность нулей и единиц.

Чтобы превратить код на Python в машинный код, используется **интерпретатор** — программа, которая читает ваш код строка за строкой и сразу выполняет.

Для Java используется **компилятор** — программа, которая переводит весь код в машинный код заранее, а потом компьютер выполняет уже готовый результат.

Вам не нужно понимать разницу между компилятором и интерпретатором в деталях (хотя это полезно). Важно знать: программист пишет код на удобном языке, а специальные программы переводят его в машинный код, понятный процессору.

## Почему это важно для аналитика

Когда заказчик говорит «хочу мобильное приложение», вам нужно понимать:

- Приложение нужно написать на каком-то языке программирования
- Для iOS обычно используют Swift, для Android — Kotlin, для кроссплатформенного — Flutter или React Native
- Выбор языка влияет на скорость разработки, стоимость и качество

Вы не выбираете язык (это делает техлид или архитектор), но вы должны понимать, что выбор есть, и у каждого варианта есть плюсы и минусы.

Ещё важный момент: **программирование — это не только написание кода**. Это ещё и проектирование архитектуры, тестирование, отладка, рефакторинг. Когда программист говорит «я пишу код», он имеет в виду весь этот комплекс задач.

## Ключевые термины

- **Программирование** — процесс создания инструкций для компьютера
- **Язык программирования** — формальный язык для написания кода
- **Компилятор** — переводит весь код в машинный код заранее
- **Интерпретатор** — переводит и выполняет код строка за строкой
- **Машинный код** — последовательность нулей и единиц, которую понимает процессор

## Проверь себя

1. Зачем нужны языки программирования, если компьютер понимает только нули и единицы?
2. В чём разница между компилятором и интерпретатором?
3. Почему выбор языка программирования важен для бизнеса?

**Ответы:** 1) Языки программирования удобны для людей, а специальные программы переводят их в машинный код. 2) Компилятор переводит весь код заранее, интерпретатор — строка за строкой во время выполнения. 3) Разные языки подходят для разных задач, влияют на скорость разработки и стоимость.
