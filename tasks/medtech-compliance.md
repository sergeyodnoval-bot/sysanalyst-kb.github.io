:::info[TL;DR]
Составить чек-лист проверки МИС на соответствие 323-ФЗ (об охране здоровья) и 152-ФЗ (ПД). Включить: требования к ЭМК, телемедицине, передаче в ЕГИСЗ, защите персональных данных. Результат: таблица требований с критериями проверки.
:::

## Контекст

МИС крупной больницы готовят к сертификации и проверке Росздравнадзора. Нужно проверить compliance с 323-ФЗ, 152-ФЗ и приказами Минздрава.

## Что нужно сделать

1. Составить перечень требований из 323-ФЗ:
   - Ведение ЭМК (ст. 22, 92)
   - Телемедицина (ст. 36.2)
   - Информированное согласие
   - Электронный рецепт
2. Составить перечень требований 152-ФЗ:
   - ПД особой категории (ст. 10)
   - Согласие на обработку
   - Локализация (ст. 18 + 242-ФЗ)
   - Уведомление РКН
3. Сформулировать критерии проверки для каждого требования
4. Определить класс системы (медизделие или нет)

## Результат

- Чек-лист (таблица: НПА → Требование → Критерий → Статус)
- Матрица ответственности (RACI) для compliance-процессов
- План устранения замечаний (gap analysis)

## Критерии приемки

- 15+ требований из 323-ФЗ и 152-ФЗ
- Для каждого — конкретный критерий проверки (например: "Есть ли согласие на обработку ПД в электронном виде?")
- Определён класс риска медизделия для МИС
- Gap analysis содержит 5+ потенциальных замечаний
